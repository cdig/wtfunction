You can write (almost) any valid CoffeeScript in the text area. You have access to global variables for time and dT, as well as a global constant TAU. Every frame, we will execute the code. The last line of your code should be a numeric value. The graph will draw that value over time. We will re-scale the graph on the fly, and show the current range of the graph as max and min in the chart above. If you want to store a variable from one frame to the next, please attach it to window (see the "Saving State" example). Undo/Redo keyboard shortcuts work when you have keyboard focus on the text area.
